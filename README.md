## ğŸ“¦ How to Run

### Install Dependencies
```bash
npm install
````

### Run File System Scripts

Uncomment the desired operation inside `fileOperations.js` and run:

```bash
node fileSystem/fileOperations.js
```


```bash
npm install -g nodemon
```
